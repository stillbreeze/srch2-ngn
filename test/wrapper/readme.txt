
There are six folders in wrapper:

1.......conf folder...........
  Some configuration files and license key file.

2.......data folder...........
  The data file which is used to create index

3.......integration folder....
  WrapperCreateIndexFromFile_Test: test the integration part.

4.......jmeter.................
  The jemter files

5.......system_tests..........
  The tests which take the srch2 eninge as a wrapper, and test the   function or queries.

6.......unit..................
  Test the unit about JSON and Kafka.
